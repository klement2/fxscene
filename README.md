                               FXScene - raytracing application
                               ================================

                               Karol Gajdos <klement2@azet.sk>

What is FXScene?
=================

FXScene is a (simple) C++ 3D modeler and raytracer.


Where to get it?
================

https://github.com/klement2/fxscene


Installation.
=============

Adjust paths to FOX Toolkit library in Makefile, run make.
